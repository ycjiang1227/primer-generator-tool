Why did the tool generate these primers?

All primer pairs generated from the input sequence are scored according to the following criteria:
  • G/C clamp: +1 for each primer
  • GC content between 40% and 60%: +1 for each primer
  • Tm between specified minimum and maximum Tm: +1 for each primer
  • Tm within 5℃ of each other: +1
The highest scoring pairs are then returned.
